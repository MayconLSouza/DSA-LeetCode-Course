# Data Structures and Algorithms (DSA) Course 

This repository contains my solutions to **LeetCode-style algorithm problems** following the **DSA course by Augusto Galego** ([Link to Course](https://app.hub.la/m/L8wi9vio7WPnWbmF8ZIO)).

## Course Overview

This course teaches fundamental Data Structures and Algorithms concepts through hands-on problem solving. Each chapter covers a specific technique or data structure, with multiple problems to reinforce understanding.

## Repository Structure

The repository is organized by **chapters**:

```
├── Arrays/
│   ├── BinarySearch/
│   ├── ContainsDuplicateII/
│   ├── ExponentialSearch/
│   ├── HashMapProblem/
│   ├── SlidingWindows/
│   ├── TwoPointer/
│   └── TwoSum/
├── README.md (this file)
└── .gitignore
```

## Problem Structure

Each problem folder contains:
- **Main.java** - Implementation of the solution
- **README.md** - Problem statement, examples, and constraints
